# 📚 Library Management System

A simple web-based Library Management System built using PHP and MySQL.

---

## 🚀 Features
- Add new books
- Delete books
- View all books
- Manage library members

---

## 🛠️ Tech Stack
- PHP
- MySQL
- HTML
- CSS
- XAMPP

---

## 📂 Project Setup
1. Clone or download the project
2. Place in `htdocs` folder (XAMPP)
3. Import database in phpMyAdmin
4. Run using:
   http://localhost/library_system/
